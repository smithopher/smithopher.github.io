VapeStack

Social network for vape enthusiasts


VapeStack

Technologies: ReactJS, JSX, AJAX, JavaScript, Bootstrap, Sass | Ruby on Rails, PostgreSQL

The idea came to me after I thought of an issue I had that I'd love to solve. After moving back to the New York City area, I had a hard time finding flavors for my vape that I really liked in the past, so I wanted to create a way to find a consensus amongst vape enthusiasts of their favorite flavors. What I came up with was VapeStack, a social network for vapers. If this were to ever go live, before I could go and market this to stores to help them identify wants amongst their client-base using data analytics, I would need to develop something that people would actually use, so I started by building the customer platform. I am probably most proud of things that I've learned through the process. I learned to conditionally render React components and how to incorporate React into a Ruby on Rails application more cohesively than I had in the past. I also learned more about making a site responsive; in previous projects I spent more time on functionality, but I wanted to make this look good across multiple devices. I would like to incorporate more ajax requests to get data, at this point the application only uses it to submit, or post, data through forms and, in one case, buttons.

Feel free to create a log-in and browse the site! You can search for people and see how your home screen populates their statuses and posts after you've added them.

Tic Tac Toe

Web application of the classic game, Tic Tac Toe


Tic Tac Toe

Technologies: HTML, CSS, SVG, JavaScript, jQuery

The classic game of Tic Tac Toe adapted to a web application. Xs and Os were created using SVG graphics. The click events are handled by jQuery to allow cross browser capability. Structure and styling are done in HTML and CSS. Game logic written in "vanilla" JavaScript.

To play, you will be prompted when landing on the page to enter a 1 or 2 for the number of players. If you enter 1, you will play against the computer (who can be pretty tricky to beat!), and entering 2 will allow you to play against a friend. If you want to change the number of players at any point, simply refresh the page. The game will automatically clear the board and prepare for another game when the alert declaring the winner is dismissed.
